US Military

Site Name

The name of this project site will be usmilitary.gov

The reason for this site name is because it represents a website giving information about the US Military. So this site name is suitable.

Site Purpose

The purpose of this site is to provide all the necessary details and information about the US Military. But the site has different sections that differ from one another talking about different branches of the military, purpose, achievements, and various jobs and their necessary requirment. Also how to enlist as a recruit or an officer.

Scenarios

What age can one join the military?

What are the basic requirments to join the military?

How can i join the military outside of the United States?

How powerful is the US military?

What are some benefits of joining the military?

Color-Schema

Color of the headers of each page will vary.

The color red will be used for the Marines header

The color black will be used for the Army header

The color navy blue will be used for the Navy header

The color aqua will be used for the Airforce header

The color of the text will be black

Typography

The fonts for the header will be Montserrat

The fonts for the paragraph will be Roboto
